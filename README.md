# Requêtez une base de données avec SQL

## Contexte et objectifs

Ce projet vise à analyser une base de données d’assurance habitation.  
L’objectif est de comprendre la structure de la base, d’écrire des requêtes SQL pertinentes et d’extraire des indicateurs utiles à la prise de décision.

## Soft Skills
- Comprendre la logique d’une base relationnelle  
- Interpréter des résultats SQL  
- Reformuler un besoin métier en requêtes techniques  
- Communiquer efficacement auprès d’un public non technique  

## Compétences
- Créer un schéma relationnel  
- Charger et interroger une base SQL  
- Écrire des requêtes incluant jointures, agrégations et filtres  
- Extraire des insights pour orienter des décisions  

## Outils utilisés
- MySQL / PostgreSQL  
- SQL Workbench ou DBeaver
